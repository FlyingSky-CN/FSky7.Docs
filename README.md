## Installation

If you don't install `pipenv`:

```shell
pip install pipenv
```

Install the dependencies:

```shell
pipenv install
```

## Development

Start development server:

```shell
pipenv run mkdocs serve
```

Build the documentation:

```shell
pipenv run mkdocs build
```
